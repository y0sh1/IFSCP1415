Opdracht 1

Dit programa geeft een hello world terug, print in de console van python.

Tests

Precondities

  • Python is opgestart
  • Opdracht 1 is ingeladen

Dit programma is te testen, door het in python uit te voeren. Het positieve resultaat zal de tekst "Hello World" terug geven.

Opdracht 2

Dit programma geeft opties weer met betrekking tot de werking van python. Door 1 van de opties in te toetsen, gevolgt door een enter op het toetsenboord, wordt de informatie weergegeven.

Tests

  • Python is opgestart
  • Opdracht 2 is ingeladen

Dit programma is te testen door, zoals beschreven, een optie te kiezen. Deze optie zal, na het indrukken van de enter toets, gegevens weergen. Bij het invoeren van iets dat niet in de opties staan, zal het systeem "Geen valide invoer. Probeer het opnieuw" terug geven.

Opdracht 3

Dit programma geeft de inhoud van een map. Bij het invoeren van een pad naar een bestand, kan deze indentifceren of dit echt een bestand is.

Tests

  • Python is opgestart
  • Opdracht 3 is ingeladen

Dit programma is te testen door een pad naar een bestaande map of een bestaand bestand in te vullen. Als je hier rechten heb om deze te kunnen bekijken zal het systeem de inhoud weergeven, of aangeven dat het een bestand is. Als het programma geen toegang heeft zal deze "Wat is dit?" terug geven

Opdracht 4

Dit programma indentificeerd alle word, excel en pdf bestanden van de opgegeven map.

Tests

  • Python is opgestart
  • Opdracht 4 is ingeladen
  • Er zijn pdf en word of excel bestanden aanwezig
  • De lokatie van deze bestanden is bekend

De gebruiker dient het bekende pad op te geven. Als deze dit gedaan heeft en vervolgens op de enter toets drukt, zal per bestand weergegeven worden welke pdf, word of excel zijn.

Opdracht 5

Dit programma leest de GPS uit de meta data van een foto, als deze beschikbaar is/zijn.

Tests

  • Python is opgestart
  • Opdracht 5 is ingeladen
  • Er zijn foto's aanwezig met gps data
  • De gebruiker weet waar deze foto's staan

Kies een pad, waarvan bekend is dat er foto's aanwezig zijn met GPS data hierin. Na het indrukken van de enter toets. Als de gebruiker het bekende pad opgeeft, waar foto's staan, zullen deze een voor een bekeken worden. De resultaten zullen op het scherm weergegeven worden.

Opdracht 6

Met dit programma is het mogelijk om een bericht te coderen. Dit programma werkt met alles in 16 voudige aantallen. Dit betekend dat zowel de sleutel, als de plaintext hiervan een 16 voudig getal moet bevatten.

Tests

  • Python is opgestart
  • Opdracht 6 is ingeladen

De gebruiker dient als eerste een sleutel op te geven. Vervolgens mag deze kiezen wat deze wilt doen. Als eerste gaan we een text coderen. kies hiervoor 1.
